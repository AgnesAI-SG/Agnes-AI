# Agnes AI Support

This repository is the public developer support and documentation hub for Agnes AI.

## Where to Ask

| Need | Use |
| --- | --- |
| API integration does not work | GitHub Issues |
| Authentication, rate limit, model, image, or video API errors | GitHub Issues |
| Documentation is unclear or missing | GitHub Issues |
| Feature requests with a clear desired behavior | GitHub Issues |
| Open-ended product questions | GitHub Discussions |
| Community examples, ideas, and best practices | GitHub Discussions |
| Announcements and roadmap conversations | GitHub Discussions |

## Before Opening an Issue

Please remove all secrets and private data before posting.

Do not include:

- API keys or bearer tokens
- Private account information
- Customer data
- Full private logs
- Screenshots containing secrets

Helpful issue reports usually include:

- Model name
- Endpoint
- SDK or client name and version
- Error code and error message
- Minimal request body with secrets removed
- Expected behavior
- Actual behavior
- Whether the issue is reproducible

## Issue Scope

Use issues for actionable items that maintainers can investigate or resolve:

- API integration problems
- Reproducible errors
- Documentation gaps
- Compatibility reports
- Confirmed feature requests

Use discussions for open-ended questions:

- Which model should I use?
- How should I structure a workflow?
- Is anyone using Agnes AI with a specific tool?
- What would be the best API design for a future feature?

## Response Expectations

This is a public repository. Public responses may summarize known behavior, request a minimal reproduction, link documentation, or close issues that are resolved in the current docs.

For account-specific billing, quota, private logs, or sensitive production incidents, use the official Agnes AI support channel instead of posting private information in GitHub.
